# Foody E-commerce website

```bash
Tech Stack

1. Next.js
2. Tailwind CSS
3. AOS Library (For animations)

```

- Foody is an ecommerce website for groceries and cereals.
- It has a landing page with a hero section, about section, why us section, product preview section, and a footer section.


```bash
bun dev
```

This will start the development server


This is a serverless web application that allows users to purchase groceries and cereals online.
